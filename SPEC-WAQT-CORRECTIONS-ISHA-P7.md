## N°845 — Argument cosmologique (SUPPRIMER — doublon n°652)

**SUPPRIMER** l'entrée complète n°845 du JSON.

```
L'univers a eu un commencement. Or tout ce qui commence à exister a une cause. L'univers a donc une cause — qui, elle, n'a pas commencé. Simple, et difficile à esquiver.
```

## N°846 — Hawking / "esprit de Dieu" (nuancer)

**REMPLACER :**
```
Tu vas dire 'mais Hawking ?' Hawking a écrit dans Une Brève Histoire du Temps : 'Si nous trouvons la théorie complète, alors nous connaîtrons l'esprit de Dieu.' Lui aussi.
```
**PAR :**
```
Tu vas dire 'mais Hawking ?' Hawking a écrit : 'Si nous trouvons la théorie complète, nous connaîtrons l'esprit de Dieu.' Il précisait plus tard que 'Dieu' était pour lui une métaphore. Mais l'émerveillement devant l'ordre du réel — il le partageait.
```

## N°847 — "Personne n'a réfuté" (reformuler)

**REMPLACER :**
```
Personne n'a jamais réfuté l'existence d'Allâh. Personne. Pas un seul philosophe, pas un seul scientifique. Le maximum qu'on a pu faire, c'est 'douter' — pas réfuter. Le doute n'est pas une preuve d'inexistence. C'est l'absence d'une preuve de l'inverse.
```
**PAR :**
```
Personne n'a jamais prouvé l'inexistence d'Allâh. Le maximum qu'on puisse faire, c'est douter — pas réfuter. L'absence de preuve n'est pas une preuve d'absence. C'est Russell, pas l'islam, qui l'a dit.
```

## N°858 — Attribution Imam Ali (tempérer)

**REMPLACER :**
```
Imam Ali a dit : 'Si Allâh n'existait pas et que je vivais selon Son commandement, je n'aurais rien perdu. S'Il existe et que je ne l'ai pas suivi, j'aurais tout perdu.' Pascal a redit cela 1000 ans plus tard et on l'a baptisé 'pari de Pascal'. L'idée venait avant lui.
```
**PAR :**
```
Un raisonnement similaire est attribué à Imam Ali : 'Si Allâh n'existait pas et que je vivais selon Son commandement, je n'aurais rien perdu. S'Il existe et que je ne l'ai pas suivi, j'aurais tout perdu.' Pascal l'a reformulé 1000 ans plus tard. L'idée précède son nom occidental.
```
