## N°1391 — Iram aux colonnes (nuancer)

**REMPLACER :**
```
Les satellites ont peut-être trouvé ses ruines. Allâh dit vrai même quand l'histoire l'oublie.
```
**PAR :**
```
Les satellites ont peut-être repéré ses traces dans le désert d'Arabie — certains chercheurs le pensent. Allâh dit vrai même quand l'histoire hésite.
```

## N°1412 — Respiration "Hû" (encadrer)

**REMPLACER :**
```
Tu peux respirer en pensant 'Hû'. À l'inspiration : Hû. À l'expiration : Hû. C'est la respiration des soufis. Essaye une fois, juste pour voir.
```
**PAR :**
```
Certains soufis respirent en pensant 'Hû' — un des Noms d'Allâh. Ce n'est pas une obligation. Juste une invitation à conscience. Essaye une fois, juste pour voir.
```

## N°1420 — Citation soufie (tempérer attribution)

**REMPLACER :**
```
Les soufis disent : 'Le silence est la langue d'Allâh, tout le reste est mauvaise traduction.' Médite ce soir.
```
**PAR :**
```
On dit dans les cercles soufis : 'Le silence est la langue d'Allâh, tout le reste est traduction.' Médite ce soir.
```
