## N°1525 — "Prière du soir" (corriger)

**REMPLACER :**
```
Dans la prière du soir, tu dis 'Iyyâka na'budu wa iyyâka nasta'în'.
```
**PAR :**
```
Dans chaque rak'a, tu dis 'Iyyâka na'budu wa iyyâka nasta'în'.
```

---

## N°1529 — "Financer une religion" (reformuler)

**REMPLACER :**
```
La femme du Prophète ﷺ Khadija a financé une religion. Puis elle est morte. Son tafwîd n'a pas attendu de voir l'islam triompher. Le tien non plus.
```
**PAR :**
```
La femme du Prophète ﷺ Khadija a donné tout ce qu'elle avait pour soutenir le Prophète ﷺ sans voir l'islam triompher de son vivant. Son tafwîd n'attendait pas le résultat. Le tien non plus.
```

## N°1558 — Narration "bourdonnement" (tempérer)

**REMPLACER :**
```
Les Compagnons disaient : 'On entendait un bourdonnement dans le ciel quand la sakîna descendait.' Tu n'entendras peut-être rien. Mais sentir suffit.
```
**PAR :**
```
On rapporte que certains Compagnons percevaient quelque chose d'inhabituel quand la sakîna descendait. Tu n'entendras peut-être rien. Mais sentir suffit.
```
