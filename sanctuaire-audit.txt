RESUME
=======
Le Sanctuaire est compose de :
  - v2GoSanctuaire() : orchestrateur principal (transitions, UI setup, banners)
  - updateSanctuaireMoment() : bandeau contextuel priere/moment (le coeur dynamique)
  - v2RefreshStats() : stats, greeting/murmure, encarts decouverte, tawba glow
  - v2UpdateOrbState() : etat de l'orbe (intention posee ou non)
  - renderLevelStripCondensed() : barre de progression niveau
  - triggerSpontaneousUI() : cascade de modales spontanees (tawba, level-up, bilan, etc.)
  - applyAtmosphere() : theme visuel base sur le mois hijri

20 overlays/modales possibles depuis le Sanctuaire.
